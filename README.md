### 😂 Bringing the laughs one joke at a time!

<div align="left">

A fun and interactive **Random Joke Generator** with an **Open Mic Night 🎙️** theme. Choose your favorite joke category, reveal the punchline when you're ready, and keep track of your latest laughs—all powered by the amazing **JokeAPI**.

---

## ✨ Features

🎭 **6 Joke Categories**
- 🌍 Any
- 🎲 Misc
- 💻 Programming
- 🤓 Pun
- 👻 Spooky
- 🖤 Dark Humor

😂 **Two-Part Jokes**
- Suspense first...
- Then click **Reveal Punchline** 🎉

📜 **Recent Joke History**
- Automatically saves your last **10 jokes**
- Scroll through your recent laughs anytime

⚡ **Loading & Error States**
- Smooth loading feedback
- Friendly error messages when something goes wrong

🛡️ **Safe by Default**

  - 🚫 NSFW
  - 🚫 Racist

  - 🚫 Religious
  - 🚫 Political jokes
- Dark Humor is available only when you intentionally select it.

📱 **Responsive Design**
- Looks great on desktop 💻

- Mobile 📲
- Supports reduced-motion accessibility ♿

## 🚀 How It Works

1️⃣ Select your favorite joke category 🎭

2️⃣ Click **Get a Joke** 😂

3️⃣ The app requests a joke from:

```
https://v2.jokeapi.dev/joke/{category}
```

4️⃣ If it's:
- 😂 One-liner → Displays immediately
- 🎤 Two-part → Shows the setup first, then lets you reveal the punchline

5️⃣ Every joke gets added to your **Recent Jokes** history.
